# gtensor-new-blog
This is a new version blog based hexo

## Installation

``` bash
$ npm install hexo-cli -g
```

## Quick Start

**Setup your blog and search plugin**

``` bash
$ hexo init blog
$ cd blog
$ npm install --save hexo-generator-search

```

**Start the server**

``` bash
$ hexo server
```

**Create a new post**

``` bash
$ hexo new "Hello Hexo"
```

**Generate static files**

``` bash
$ hexo generate
```
